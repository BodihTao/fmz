灵界故事之御剑飞行梦

By: 分明子 原文发布于：2019-08-22 06:57:40


         人的执念，就想海水一样，取之不尽的。一位散发红色火焰的天将军护法现身，如是说道。今天我为有缘讲一个本生故事。


           前生，我出生在一个平常的普通农家，在我幼年时，父母让我好好读书，考取功名，出人头地，光宗耀祖。当我十八岁时，有缘看到一本神话故事书，主要是讲吕祖吕洞宾，御剑飞行等仙术，心神向往，便痴迷其中。


         从此以后，如痴如醉，沉迷其中不能自拔。某天，为实现这御剑飞行的梦想，便偷偷跑出去，到处寻访仙师。也算机缘巧合，来到一座大山中，看到一位中青道士，当时我就想，要学仙术，先成为道士，书中的故事都是这样，便拜在门下求学道术。


          自从入门后，师父只教我打坐练气，念经等事。一年后，我实在有点憋不住了，便去求问师父，我这样修，能不能修成仙术？师父回答道，不能。当时，我心里很失望，梦想打碎，再留下来没意义，便偷偷溜下山，想找一位高人学仙术。


         随后，我到处游访，某天听说有一位高人会仙术，便去拜访求教，结果此高人一口否认，他没有仙术，不会御剑飞行。我是真心求教，他是真心推辞，并呵斥我，世上那有什么仙术，修道人应该好好修道，别想着走捷径，我心有不服，便争论起来，最后我俩大打出手，无意间，我当场打死高人。


          发生人命案，要吃官司，以命尝命，心中极度恐惧，我不想做牢，更不想死，赶紧逃离了现场，直往曾经学道的山里跑。刚跑到山门口，又看到打死的高人安祥地坐在门槛上，好像特意在等我似的。吓得我赶紧掉头准备逃走，结果师父突然出现，并笑着对我说道，不用害怕，这只是我的一个分身。这么神奇，吓的我当时跪到师父面前忏悔。师父又道，看你如此执着仙术，好吧，你按我的方法修，一定会得到仙术的。师父还是要求我每天打坐念经，当然，还有密法口诀，每天修炼不得中断。


          十年后，仙术小成，我真的可以御剑在空中自由飞行，梦想成真，便告别师父，准备下山去。师父当时也没有挽留，只是笑了一下，回答道，你去吧！来到红尘俗世中，有仙术在身，便开始炫耀自己的仙术，准备御剑飞行。当一切准备好时，开始御剑飞行，结果，熟练驾驭的仙剑却飞不起来。颜面丢尽，羞愧难当，便再次反回深山，求教师父。


          我重回山中，对师父说道，为什么我在山上练得这么熟，在红尘中却飞不起来？师父似乎知我所疑，只笑了一笑，反问道，学道为了什么？我答：学道为了仙术。师父道，徒弟你错了，学道是为了脱生死，只要了脱生死，你想要什么样的仙术，都会有，到时候，万般神皆小求，还求什么仙术？听了这段话后，恍然大悟，决定先了脱生死，便又留下来继续跟师父修练。


         三十年后，我也近六十多岁，师父也八十多岁了。某天，师父把我叫到面前，问道，还想不想学仙术了？当时，我已看明白这些情况，爽快的回答道，不想了，现在只想了脱生死。师父开心的笑了笑，说道，徒弟啊！我时日不多了，你多陪我一下，待我升天后，你就下山去吧！不要留在山上，到红尘去，行善累积功德。


        几月后，师父把我叫到面前，说道，我世缘已尽，我要走了，我走后，你按我的方法好好修炼。说完后，双腿一盘，神奇的一幕出现了，师父慢慢融化，散发红光，在红光中，化的一无所有，怎么都没有留下。


         师父升天后，我信心巨增，遵师命，来到红尘俗世中，练道修心，累积功德。我在红尘中帮人，本已修得神通，所以，一般情况下，先观其因果，看能不能帮，如何帮，再做决定。常做的治病救人，解难消灾，由于度化有缘。


         某天，我又遇到师父，当时，很惊讶，便问道，师父你老人家已羽化飞升了，怎么又出现了？师父道，那只是我的肉体消亡，其实我从没离开过你，现在你也可以升天了，但是必须舍弃肉身，才可以升天，升天后，自有你的安身之处。


        听了师父之话后，我自知世缘已尽，

便找了一隐蔽地方，盘腿而坐，元神出窍，脱离了肉体。此时，天空出现很多天人，一同把我迎上天上。到天上后，也无所事事，又过了一段时间，玉皇大帝传下玉旨，安排我到某处某家去，此道场现在缺护法，让我去做护法。想到师父的话，此处应该就是我的安身之处，从此便安安心心，入驻道场，直到现在。


         万般神通是小术，

         唯有成道方是真，

         御剑飞行南柯梦，

         跳出三界自成圣。

有求皆苦，心无所求无挂障，没有恐怖，自然远离颠倒梦想，必将得证涅槃。谢谢您讲故事！


           我之瞎掰系列，一茶饭笑谈，当故事看，看里面的料，不要执着真假，更不能做圣解，也敬请勿谤勿辩，学佛以戒为师，经教为师，应诸恶莫作，众善奉行。同时，自以为是的人，免费看故事，不要你感恩，不喜欢就远离，留言何用？不必刷存在感，尊重别人，等同尊重你自己。原创因果灵异故事，记录修学心得，爱国，爱教，爱家，破除迷信邪见，拒绝烧纸钱元宝，传播道德正能量。
